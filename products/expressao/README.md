# Expressão

Sistema para transformar mensagens em conteúdo com forma específica.

## Posição no Pipeline
```
Camadas da Linguagem → Expressão → Conteúdo
      (Mensagem)        (forma)
```

## O Que Faz

- Recebe Mensagem estruturada (de Camadas da Linguagem)
- Transforma em conteúdo com forma específica
- Formas: post, email, apresentação, etc.
