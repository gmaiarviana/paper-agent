# Camadas da Linguagem

Sistema para estruturar ideias em mensagens comunicáveis.

## Posição no Pipeline

```
Revelar/Prisma Verbal → Camadas da Linguagem → Expressão
        (Ideia)              (Mensagem)         (Conteúdo)
```

## O Que Faz

- Recebe Ideia estruturada (de Revelar ou Prisma Verbal)
- Organiza argumentos e evidências conforme intenção
- Produz Mensagem pronta para ser "materializada"

## O Que NÃO Faz

❌ Não produz conteúdo final (isso é Expressão)  
❌ Não cria clareza de ideias (isso é Revelar)  
❌ Não extrai conceitos de textos (isso é Prisma Verbal)
