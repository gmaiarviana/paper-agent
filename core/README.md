# Core Universal

Sistema multi-agente universal que serve múltiplos produtos.

## Componentes

- **Agentes**: Orquestrador, Observador, Estruturador, Metodologista, Memory Agent
- **Ontologia**: Conceito, Ideia, Argumento, Proposição, Evidência
- **Infraestrutura**: LangGraph, ChromaDB, SQLite
